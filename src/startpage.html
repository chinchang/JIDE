JIDE - Java Integrated Development Environment


GETTING STARTED

WRITING YOUR CODE
  1. Create a new file from the "File" menu.
  2. Write your Java code.
    Sample JAVA code : 
    class testrun{
	public static void main(String args[]){
		System.out.println("Hello World!");
	}
}
  3. Save the file from the "File" menu.
    NOTE: Save the file with the name of your class.


SETTING THE JDK PATH
  1. Open "Compiler Profiles" from the "Options" menu.
  2. Paste the location where your JDK is installed.
    3. For Example :
    If its installed in C:>Program Files>Java>jdk1.6.0_14, then paste C:\Program Files\Java\jdk1.6.0_14 in the text box.
  3. Click on "Save" to save the settings.


COMPILING & RUNNING YOUR CODE
  1. To Compile, select "Compile" from "Build" menu. The errors, if any, get displayed in the bottom Output Window.
  2. Once your code compiles without any errors, run it by going to "Run" in "Options" menu. See the program output in the Output Window.
